Getting Started
01
Open the file
imageplot_2.html — open in any modern browser (Chrome, Firefox, Safari). No installation or server required.
02
Load a data file
Click Data File and choose a .tsv or .txt file (tab-separated). The app auto-detects numeric columns.For the filename column the app looks in priority order: a column named exactly filename, then any non-numeric column whose name contains "file", "name", or "image", then the first non-numeric column, and finally the first column of any type as a fallback. A year column is used for color by default.
03
Load an image folder
Click Image Folder and select the folder containing your images. Supported formats: JPG, PNG, GIF, WebP, TIFF. Filenames must match the filename column in your data file (case-insensitive).
04
Extract features (optional)
Choose Basic (6) or All (77) features, then click Extract. Extracted features are combined with the columns in your data file and can be exported as TSV for reuse.
05
Visualize
Set X and Y axes from any numeric column, or run PCA / t-SNE across selected features. Switch between Images and Circles display modes.
06
Hide the panel
Click the slim ‹ tab on the left edge of the canvas to collapse the control panel and give the visualization full-width view. Click again to bring it back.
Data File Format
Tab-separated values, one row per image. The first row must be headers. Any all-numeric column can be used in visualization. The filename column is recognized automatically if named filename, file, name, or image.

Workflow Tips
Extract features once, export as TSV, then reload that TSV on future sessions — avoids re-extracting each time.
For large collections (500+), use Basic extraction and PCA first. Switch to All (77) features and t-SNE for refined analysis.
The Suggest axes link scores all column pairs by k-means silhouette — click a suggestion to apply it instantly.
Visualization Modes
X / Y Axes
Choose any two numeric columns.When your data has four or more numeric columns, use Suggest axes: the app analyzes all possible pairs of features — running k-means clustering on each pair and scoring how distinctly the images separate into groups — then ranks and lists the pairs that show the clearest cluster structure. Click any suggestion to apply it instantly.
PCA
Positions each image in 2D space along the directions of greatest variation across your selected features. Images with similar visual characteristics appear closer together. Select features using the checklist, then click Run PCA.
t-SNE
Positions each image in 2D space using nonlinear dimensionality reduction. Usually this method is better for separating clusters of similar images. Tune Perplexity (2–50; try 15 for small collections) and Iterations (100–1500). Click Run t-SNE. Results vary between runs.

Display & Animation
Images mode — renders each image as a thumbnail. Adjust Thumbnail size (16–200 px) as you see fit. Larger thumbnails are more legible; smaller are better for showing patterns.
Circles mode — draws colored circles. Circle size scales with the color value; adjust base radius (3–30). A gradient legend appears top-right.
Color by — maps any column to a blue→teal→amber→red gradient. Defaults to year if present.
Labels — enable this option to show any column value above each point. Set font size (7–24 px) and color independently.
Animation — choose a column and click Animate. Points appear one by one sorted by that column's value. A running count and current value display in the plot. Click Stop to halt.
