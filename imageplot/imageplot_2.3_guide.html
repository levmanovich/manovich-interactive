Getting Started
01
Open the file
imageplot_2.2.html — open in any modern browser (Chrome, Firefox, Safari). No installation or server required.
02
Two ways to begin
A · Start from data — load a .tsv/.txt file and visualize its numeric columns directly.B · Start from images — load an image folder on its own; the app now auto-extracts Basic features and plots them in one step — no data file required.
03
Load a data file
Click Data File and choose a tab-separated file. The app auto-detects numeric columns. For the filename column it looks, in order, for a column named exactly filename, then any non-numeric column containing "file", "name", or "image", then the first non-numeric column, then the first column. A year column is used for color by default.Loading a data file keeps any images you already loaded and merges the new columns with previously extracted features (matched by filename).
04
Load an image folder
Click Image Folder and select the folder of images. Supported: JPG, PNG, GIF, WebP, TIFF. Filenames are matched to the data file's filename column (case-insensitive). With no data file present, extraction runs automatically once the folder finishes loading.
05
Extract features
Choose a tier — Basic (6), Standard (21), or All (80) — then click Extract (runs automatically for the images-only path). That tier is the feature set the visualization uses. Extracted features merge with your data file's columns and can be exported as TSV for reuse.
06
Visualize
Set X / Y axes from any numeric column, or run PCA / t-SNE / UMAP across the extracted features. Optionally narrow a projection to one Feature family (Grayscale / Color / Structure), and switch between Images and Circles display.
07
Hide the panel
Click the slim ‹ tab on the left edge of the canvas to collapse the control panel for a full-width view. Click again to restore it.
Data File Format
Tab-separated values, one row per image. The first row must be headers. Any all-numeric column can be used in visualization. The filename column is recognized automatically if named filename, file, name, or image.

Workflow Tips
Drop an image folder in on its own for a quick look — Basic features extract and plot automatically. Add a data file later to bring in metadata like year; your images stay put.
Extract once, export as TSV, then reload that TSV in future sessions to avoid re-extracting.
For large collections (500+), start with Basic or Standard + PCA, then move to All (80) with t-SNE or UMAP for refinement.
The Suggest axes link scores column pairs by k-means silhouette and ranks those with the clearest separation — click a suggestion to apply it.
Feature Tiers — chosen at Extract
Basic (6)   Standard (21)   All (80)
Basic = brightness / saturation / hue median & stdev. Standard adds the rest of the directly interpretable features — entropy, edges, contrast, aspect ratio, RGB means, plus symmetry, dominance and weight_offset (21 total). All adds the technical features used mainly by dimension reduction: the 3×3 spatial grid and the grayscale / saturation histograms (80 total).Pick the tier once, at extraction — that is the whole feature set the visualization uses. There is no separate feature-selection step. To refine a projection, optionally tap a Feature family chip (see Visualization Modes).
Visualization Modes
X / Y Axes
Choose any two numeric columns. With four or more numeric columns, Suggest axes runs k-means on every pair and ranks those with the clearest cluster structure — click one to apply.
PCA
Places each image along the directions of greatest variation across the extracted features; similar images sit closer. Deterministic and reproducible. Just click Run PCA.
t-SNE
Nonlinear projection that often separates clusters more sharply. Tune Perplexity (try 15 for small sets) and Iterations. Results vary between runs; cluster size and distance are not meaningful.
UMAP
Nonlinear projection that tends to preserve global structure a little better than t-SNE. Tune Neighbors, Min dist, and Epochs. Like t-SNE, treat the layout as a view, not a metric.
Feature family (optional)
Under PCA / t-SNE / UMAP, tap Grayscale, Color, or Structure to project on just that family of features; tap All (the default) to use every extracted feature again. The status line shows what's in play.
Show clusters
In PCA / t-SNE / UMAP, overlay k-means cluster ellipses. Pick a fixed k (2–5) or Auto (best silhouette). Tip: for a specific question, pin k — Auto maximizes separation, not your intent.

Display & Animation
Images mode — renders each image as a thumbnail. Adjust Thumbnail size (16–200 px): larger is more legible, smaller reveals patterns.
Circles mode — draws colored circles sized by the color value; adjust base radius (3–30). A gradient legend appears top-right.
Color by — maps any column to a blue→teal→amber→red gradient. Defaults to year if present.
Labels — show any column value above each point, with independent font size (7–24 px) and color.
Animation — choose a column and click Animate; points appear one by one sorted by that column, with a running count. Click Stop to halt.
